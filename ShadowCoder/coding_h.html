Coding (Higher Level)

📌 What is Coding?

Coding is the process of writing instructions (programs) that tell a computer what to do.

⚙️ Key Concepts

  • Input: Data given to a program
  • Output: Result produced by a program
  • Variables: Store data (e.g. age = 15)
  • Data types: Integer, String, Boolean

🔀 Selection (IF statements)

IF age >= 18 THEN
    PRINT "Adult"
ELSE
    PRINT "Minor"
END IF

Selection allows a program to make decisions.

🔁 Iteration (Loops)

FOR Loop

FOR i = 1 TO 5
    PRINT i
NEXT i

WHILE Loop

WHILE x < 10
    x = x + 1
END WHILE

🧠 Algorithmic Thinking (HL Focus)

  • Breaking problems into steps
  • Writing logical instructions
  • Debugging errors in code

🧪 Exam Questions

Q1: What is a variable? (3 marks)

Answer: A variable is a named storage location used to hold data in a program.

Q2: Explain the purpose of an IF statement. (4 marks)

Answer: It allows a program to make decisions based on conditions.

Q3: Describe a loop in programming. (4 marks)

Answer: A loop repeats a set of instructions until a condition is met.

🎯 Exam Tips

  • Always define variables clearly
  • Mention “decision-making” for IF statements
  • Use words like “repeat” and “condition” for loops
  • Show understanding, not just memorised code
