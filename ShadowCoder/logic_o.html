Logic Gates (Ordinary Level)

📌 What are Logic Gates?

Logic gates are simple electronic components that take inputs (0 or 1) and give an output based on a rule.

⚙️ Basic Logic Gates

AND Gate

Output is 1 only if BOTH inputs are 1.

OR Gate

Output is 1 if at least ONE input is 1.

NOT Gate

Changes 1 to 0 and 0 to 1.

💡 Example

If a light turns on only when TWO switches are on, that is an AND gate.

🧪 Exam Questions

Q1: What is a logic gate? (3 marks)

Answer: A device that takes binary inputs and produces an output based on a rule.

Q2: What does an OR gate do? (3 marks)

Answer: It outputs 1 if at least one input is 1.

Q3: Give one example of a logic gate in real life. (3 marks)

Answer: A security system or light switch system.

🎯 Exam Tips

  • Learn simple definitions only
  • Focus on AND, OR, NOT
  • Use real-life examples like lights or alarms
  • No need for combination circuits in OL
