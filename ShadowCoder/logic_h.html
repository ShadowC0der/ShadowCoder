Logic Gates (Higher Level)

📌 What are Logic Gates?

Logic gates are electronic components that take one or more binary inputs (0 or 1) and produce a single output based on a logical operation.

⚙️ Basic Logic Gates

AND Gate

Output is 1 only if BOTH inputs are 1.

OR Gate

Output is 1 if at least ONE input is 1.

NOT Gate

Inverts the input (1 becomes 0, 0 becomes 1).

🔗 Combination Logic (HL)

Combination logic uses multiple gates together in one circuit.

Example:

(A AND B) OR C

You solve it step-by-step:

  • First solve AND gate
  • Then apply OR gate to result

🧪 Exam Questions

Q1: What is a logic gate? (3 marks)

Answer: A device that takes binary inputs and produces an output based on a logical rule.

Q2: Explain the AND gate. (4 marks)

Answer: The AND gate outputs 1 only when both inputs are 1. Otherwise it outputs 0.

Q3: Describe how to solve a combination circuit. (6 marks)

Answer: Solve each gate step by step, starting with inner brackets, then move outward until final output is found.

🎯 Exam Tips

  • Always do truth tables step-by-step
  • Start inside brackets first in combinations
  • Clearly label inputs and outputs
  • Show working — marks are for method, not just answer
