Coding (Ordinary Level)

📌 What is Coding?

Coding is writing instructions that tell a computer what to do.

⚙️ Basic Concepts

  • Input: Data you give the program
  • Output: Result from the program
  • Variable: Stores information (like age or name)

🔀 IF Statement (Decision)

IF age >= 18 THEN
    PRINT "Adult"
ELSE
    PRINT "Child"
END IF

This lets a program make decisions.

🔁 Repetition (Basic Idea)

A loop repeats instructions. 

Repeat 5 times:
    Print "Hello"

💡 Example

A login system checks if password is correct.

  • If correct → access granted
  • If wrong → access denied

🧪 Exam Questions

Q1: What is coding? (3 marks)

Answer: Writing instructions for a computer to follow.

Q2: What is a variable? (3 marks)

Answer: A variable is used to store data.

Q3: What does an IF statement do? (3 marks)

Answer: It allows a program to make decisions.

🎯 Exam Tips

  • Keep answers short and simple
  • Use everyday examples like login systems
  • Don’t overcomplicate loops
  • Focus on understanding, not syntax
