Algorithms (Higher Level)

📌 What is an Algorithm?

An algorithm is a step-by-step sequence of instructions used to solve a problem or complete a task. It must be clear, precise, and unambiguous.

⚙️ Characteristics of an Algorithm

  • Inputs (data goes in)
  • Outputs (result comes out)
  • Definiteness (clear steps)
  • Finiteness (must end)
  • Effectiveness (must work in real time)

💡 Example Algorithm (Find largest number)

Step 1: Start
Step 2: Take three numbers
Step 3: Compare first and second
Step 4: Store larger value
Step 5: Compare result with third number
Step 6: Output largest number
Step 7: Stop

🔁 Types of Algorithms

Linear Search

Checks each item one by one until found.

Binary Search

Works on sorted data by dividing search space in half each time.

⏱ Efficiency (HL Important)

Efficiency means how fast and resource-friendly an algorithm is.

  • Linear search → slower (O(n))
  • Binary search → faster (O(log n))

🧪 Exam Questions

Q1: Define an algorithm. (3 marks)

Model Answer:
A step-by-step set of instructions used to solve a problem in a finite number of steps.

Q2: Explain why binary search is more efficient than linear search. (6 marks)

Model Answer:
Binary search divides the dataset into halves each time, reducing comparisons, while linear search checks each item one by one.

🎯 Exam Tips (VERY IMPORTANT)

  • Always mention “step-by-step” in definitions
  • For HL, mention efficiency (O notation if possible)
  • Use correct comparison terms: faster / slower / comparisons
  • For full marks, always explain “why”, not just what
