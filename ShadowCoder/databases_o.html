Databases (Ordinary Level)

📌 What is a Database?

A database is an organised collection of data stored on a computer so it can be easily accessed and updated.

📊 Structure of a Database

  • Table: A collection of data
  • Record: One row of data (one person/item)
  • Field: One piece of information (e.g. name)

🔑 Keys

Primary Key

A field that uniquely identifies each record.

Foreign Key

A field used to link two tables together.

💡 Example

In a school database:

  • StudentID → Primary Key
  • StudentID in another table → Foreign Key

🧪 Exam Questions

Q1: What is a database? (3 marks)

Answer: An organised collection of data stored on a computer.

Q2: What is a primary key? (3 marks)

Answer: A field that uniquely identifies each record in a table.

Q3: Give one example of a database. (3 marks)

Answer: School database or library system.

🎯 Exam Tips

  • Keep definitions short and clear
  • Use simple examples like school or library
  • Always mention “uniquely identifies” for primary key
  • No need for advanced concepts like normalisation
